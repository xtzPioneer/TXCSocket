## 介绍

基于‘CocoaAsyncSocket’所开发的客户端Socket。

## 运行示例

要运行示例项目，请克隆存储库，然后首先从Example目录运行`pod install`。

## 要求

iOS 8.0+

## 安装

可通过CocoaPods获得TXCSocket。要安装它，只需将以下行添加到您的Podfile中：

```ruby
pod 'TXCSocket'
```
